# Privacy Policy

**Legendary Ninja: Shadow Path** does not collect, store, or share any personal user data.

## Data Collection
The game stores game progress locally on the user’s device using local storage technologies (IndexedDB).
This data is used only to save the game state and is never transmitted to any server.

## Information We Do Not Collect
We do not collect:
- Names
- Email addresses
- Location data
- Contacts
- Photos or media files
- Device identifiers

## Third-Party Services
The game does not use third-party services that collect user data.

## Children’s Privacy
The game does not knowingly collect any personal information from children under the age of 13.
